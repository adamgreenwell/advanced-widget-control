Plugin adds additional widget options for better widget management, placement and manipulation. It uses the "classic"
widget editor currently with future plans to accommodate the block-based widgets editor.

== Installation ==

1. In your admin panel, go to Plugins > and click the Add New button.
2. Click Upload Plugin and Choose File, then select the plugin .zip file. Click Install Now.
3. Click Activate to use your new plugin right away.
